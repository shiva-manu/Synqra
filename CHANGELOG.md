# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0-alpha] - 2026-02-20

### Added
- Initial alpha release
- Query Builder API with support for SELECT, INSERT, UPDATE, DELETE
- Multi-database support (MongoDB and PostgreSQL)
- Schema abstraction with `db.model()` API
- Model CRUD operations (find, create, update, delete)
- Schema validation using Zod
- Default values support for model fields
- Transaction support
- Query operators (eq, gt, lt, gte, lte, in)
- Query features (limit, orderBy, select/projection)
- TypeScript type definitions
- Comprehensive documentation and examples

### Features
- **Query Builder**: Fluent API for building database queries
- **Model System**: Define schemas with validation and defaults
- **Multi-Database**: Same API works with MongoDB and PostgreSQL
- **Type Safety**: Full TypeScript support
- **Validation**: Automatic schema validation using Zod
- **Transactions**: Support for database transactions

- MongoDB adapter examples
- PostgreSQL adapter examples
- Query builder examples

- Transaction examples

- Comprehensive README with quick start guide
